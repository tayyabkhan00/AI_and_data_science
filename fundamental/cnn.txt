‚≠ê 2. CNN solves this by using Convolutions

‚≠ê 3. How Convolution Works (Very Easy)
Example kernel (edge detector):
1  0 -1
1  0 -1
1  0 -1
This detects vertical edges.
When this kernel slides across the image:
If pixel region matches ‚Üí output positive value
If opposite ‚Üí negative value
If flat area ‚Üí near zero
This is how CNN sees shapes.

‚≠ê 4. After Convolution ‚Üí Max Pooling
Pooling reduces the image size while keeping important information.
Max Pooling example (2√ó2):

[2, 3]
‚Üí output = 5 (maximum)
This helps:
Reduce computation
Prevent overfitting
Keep only strongest signals

‚≠ê 5. CNN Architecture (Simple)

This is the heart of all image models.
