Gradient Descent Visualizer

See how gradients guide neural networks during training. The gradient points to the steepest downhill direction, helping the network adjust its weights to minimize error (loss).

Loss Surface (error)

The 3D surface represents the loss function - how wrong the network is. The height shows the error amount.

The yellow ball is the current weights of the network. We want to move it to the lowest point (minimum error).

The red arrow shows the gradient - pointing to the steepest uphill direction. We move opposite to it (downhill) to reduce error.

How Backpropagation Works

1. Network makes a prediction → Calculates error (loss)

2. Backpropagation calculates gradients for each weight

3. Gradients show how to change weights to reduce error

4. Weights are updated: new_weight = old_weight - learning_rate × gradient

5. Repeat many times until error is minimized
